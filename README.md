2. What is AWS re/Start?

AWS re/Start is a full-time training program focused on building the technical and professional skills required for entry-level cloud roles, such as:

 Cloud Support Associate
 Cloud Operations Engineer
 Technical Support Specialist
 Junior Cloud Engineer

The program combines:

 Cloud computing fundamentals
 Hands-on labs in AWS
 Programming basics (Python and Linux scripting)
 IT infrastructure knowledge
 Professional and teamwork skills


3.  What I Will Learn During the 12 Weeks

3.a. Cloud Foundations

Understanding the building blocks of cloud computing:

 Compute resources
 Storage
 Networking
 Databases
 Security fundamentals

I will also learn how these concepts translate into AWS services used by companies worldwide.


3.b.  AWS Core Services

Hands-on experience with key AWS tools such as:

 EC2 (virtual servers)
 S3 (cloud storage)
 IAM (identity and access management)
 VPC (network architecture)
 Cloud monitoring and operations tools


3.c. Programming Fundamentals

Introduction to Python and Linux scripting to:

 automate tasks
 manage infrastructure
 interact with cloud systems


3.d. Cloud Operations and Automation

Learning how cloud environments are:

 deployed
 configured
 monitored
 maintained

This includes automation practices used by modern technology teams.


3.e. Professional Skills

A key part of AWS re/Start is also professional development, including:
 communication
 teamwork
 structured problem solving
 technical thinking
 interview preparation
 building a professional digital presence

The program encourages a mindset focused on learning, curiosity, high standards, collaboration, and innovation.
