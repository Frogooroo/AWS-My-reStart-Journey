<h2>2. What is AWS re/Start?</h2>

<p>
AWS re/Start is a full-time training program focused on building the technical and professional skills required for 
entry-level cloud roles, such as:
</p>

<ul>
  <li>Cloud Support Associate</li>
  <li>Cloud Operations Engineer</li>
  <li>Technical Support Specialist</li>
  <li>Junior Cloud Engineer</li>
</ul>

<p>The program combines:</p>

<ul>
  <li>Cloud computing fundamentals</li>
  <li>Hands-on labs in AWS</li>
  <li>Programming basics (Python and Linux scripting)</li>
  <li>IT infrastructure knowledge</li>
  <li>Professional and teamwork skills</li>
</ul>
